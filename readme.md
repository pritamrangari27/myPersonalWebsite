# Pritam Rangari - Portfolio Website

A modern, responsive portfolio website for Pritam Rangari, a 3rd-year Information Technology Engineering student at AISSMS Institute of Information Technology, Pune.

## 🌟 Features

- **Modern Design**: Clean, professional layout with gradient backgrounds and smooth animations
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth scrolling, hover effects, and animated counters
- **Professional Sections**:
  - Hero section with animated typing effect
  - About me with statistics
  - Skills showcase with categorized technologies
  - Featured projects portfolio
  - Education timeline
  - Contact form with validation

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive functionality and animations
- **Font Awesome**: Professional icons
- **Google Fonts**: Poppins font family for modern typography

## 📁 Project Structure


## 🚀 Getting Started

1. **Clone or Download** the portfolio files to your local machine
2. **Replace Placeholder Images**: Add your own photos in the `images/` folder
3. **Customize Content**: Update the HTML file with your personal information
4. **Open in Browser**: Simply open `index.html` in any modern web browser

## 📝 Customization Guide

### Personal Information
Update the following sections in `index.html`:

1. **Hero Section**: Change name, title, and description
2. **About Section**: Update bio and statistics
3. **Skills Section**: Add/modify your technical skills
4. **Projects Section**: Replace with your actual projects
5. **Education Section**: Update with your academic background
6. **Contact Section**: Add your real contact information

### Images
Replace the placeholder images with your own:
- `profile.jpg` - Professional headshot (400x400px recommended)
- `about.jpg` - Additional photo (300x400px recommended)
- `project1.jpg`, `project2.jpg`, `project3.jpg` - Project screenshots (600x400px recommended)

### Colors and Styling
Modify the CSS variables in `styles.css` to match your preferred color scheme:
- Primary colors: `#3498db`, `#667eea`, `#764ba2`
- Accent colors: `#ff6b6b`, `#ffd93d`
- Text colors: `#2c3e50`, `#7f8c8d`

## 📱 Responsive Design

The portfolio is fully responsive with breakpoints at:
- Desktop: 1200px and above
- Tablet: 768px to 1199px
- Mobile: Below 768px
